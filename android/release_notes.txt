December 12, 2023
v0.3.1

🚀 Feature Updates

- System Navigation Bar 색상도 테마에 바뀌게끔 수정
- 상품 상세 화면에서 Graph 영역이 세로로 슬라이드할 경우 스크롤 영역에 적용되도록 변경 (이제부터는 길게 누를 시 그래프 라벨이 활성화됩니다)
- 일부 페이지에서 접근성 개선

🔧 Bug Fixes

- 상품 목록 화면에서 알림 토글을 껐다가 켰을때 활성화 상태에서 꺼짐 상태에 아이콘이 뜨는 오류 수정
- 상품 목록 화면에서 화면의 토글값을 변경한 후, 새로고침을 했을때 원래대로 돌아오는 문제 수정
- 로딩중일때 상품 상세 화면에서 버튼들 비활성화
- Dynamic Theme을 지원하지 않는 OS의 경우 해당 테마 설정 View 비활성화
- 상품, 상품 추천 목록 전환시 스크롤 / 당겨서 새로고침 오류 수정
- 화면 회전 시 다이얼로그가 사라지는 현상 수정
- 상품 상세 페이지에서 화면 회전 시 Scroll State가 초기화되는 현상 수정
- 모든 페이지에서 버튼 연속 입력 방지 처리
- 모든 페이지에서 메모리 누수가 일어나는 일부 항목들 수정
- 시스템 글꼴을 키웠을때 UI가 깨지는 문제 수정
- 테마 변경 후 앱 재시작 시 테마 변경이 안되는 오류 수정
- 목표 가격 수정 시 가격 수정 화면 초기 값 기존 목표 가격으로 수정
- 내 정보 페이지에서 이름에 문자만 나올 수 있도록 수정
